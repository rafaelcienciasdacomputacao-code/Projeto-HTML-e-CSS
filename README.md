# Projeto-HTML-e-CSS
A GameZoneST é uma loja virtual fictícia desenvolvida como projeto acadêmico utilizando apenas HTML e CSS. O objetivo do projeto é simular a interface de um e-commerce voltado para o universo gamer, explorando conceitos de estruturação de páginas web e estilização de layouts. O site apresenta uma vitrine de produtos como jogos, consoles e acessórios gamer, organizados em seções visuais com foco em usabilidade e estética. Todo o projeto foi construído de forma estática, sem uso de linguagens de programação ou funcionalidades dinâmicas. Este trabalho tem como finalidade praticar o desenvolvimento de interfaces web, aprimorar habilidades com HTML e CSS e criar um layout moderno inspirado em lojas online reais.
