# Projeto-HTML-e-CSS
Este projeto consiste no desenvolvimento de um site simples de uma loja de artigos esportivos utilizando apenas as linguagens HTML e CSS. O objetivo principal é simular uma loja virtual de forma organizada e funcional, mesmo sem o uso de tecnologias mais avançadas como JavaScript ou bancos de dados. A página é estruturada em diferentes seções, como página inicial, camisetas de times nacionais e internacionais, acessórios esportivos e uma linha voltada ao público feminino, com roupas de treino e tênis de corrida. A navegação é feita por meio de um menu fixo, que direciona o usuário para cada parte do site através de âncoras internas.O projeto também apresenta uma seção de produtos, onde é possível simular a visualização de itens individuais dentro da mesma página, proporcionando uma experiência semelhante à de sites de lojas reais.O principal foco deste trabalho é demonstrar o uso de estruturação em HTML e estilização com CSS, além de organizar um site de forma clara, visualmente agradável e intuitiva para o usuário.
